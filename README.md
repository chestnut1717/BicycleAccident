## 파일 설명

3. 데이터 : data 폴더 내의 데이터

 - 3.4  final_data(**실제로 사용한 데이터**) : 각종 전처리, 크롤링 데이터와 통합작업 후의 최종 excel 파일
 - 3.5  wordcloud_data : 워드클라우드 작업할 때 모아둔 폰트들
